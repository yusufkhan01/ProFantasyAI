# Screenshots

Drop a screenshot of the running dashboard here as `dashboard.png`, then enable the
image reference in the root [`README.md`](../../README.md#screenshots).

Quick way to capture one:

1. Start the app (`docker compose up --build`, or run the backend and `npm run dev`).
2. Open the dashboard, wait for the squad to load.
3. Take a screenshot and save it as `docs/screenshots/dashboard.png`.
